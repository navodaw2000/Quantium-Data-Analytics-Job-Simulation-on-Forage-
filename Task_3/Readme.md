## Chips Category Overview
- **Category:** Snack Foods – Chips  
- Peak transactions occur before Christmas  
- Promotional displays and Gondola ends increase sales  
- Key shoppers:  
  - Young Singles & Couples (largest proportion)  
  - Young & Older Families (26% of shoppers, larger basket sizes)
